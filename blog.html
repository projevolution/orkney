New Website

Unconfirmed @ 00:00


To continue developing the identity of Project Evolution, we have decided to move away from third pary resources as building our own website will give us more freedom and flexibility with layouts, content and styling which will allow us to create a more seemless design.


This website is made possible by the open source CSS resource Open-Props. This resource provides powerful, consistent and modern styles extremely easy and adaptive to your device's theme.

Along with this redesign, a more modern set of WebApp templates will be developed and released to allow for a more consistent experience.


Please note that the development of this site may take some time as it is a large project to develop. Thank you for your patience!
